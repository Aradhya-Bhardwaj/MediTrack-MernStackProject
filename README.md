# MediTrack - Medical Appointment Management System

MediTrack is a premium, full-stack medical appointment management platform built using the MERN stack (MongoDB, Express, React, Node.js). It provides a seamless experience for both patients and healthcare providers to manage appointments, profiles, and schedules.

## 🚀 Features

- Role-Based Access Control**: Separate dashboards for Patients and Doctors.
- **Appointment Management**: Patients can book appointments, and doctors can manage their schedules.
- **Premium UI/UX**: Modern glassmorphism design with a clean, professional aesthetic.
- **Authentication**: Secure JWT-**based authentication with support for Google OAuth.
- **Real-time Notifications**: Toast notifications for user feedback.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite, Vanilla CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Passport.js, JWT

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Aradhya-Bhardwaj/MediTrack.git
   cd MediTrack
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Create a .env file with your credentials
   npm start
   ```

3. **Frontend Setup**:
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
